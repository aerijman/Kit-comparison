Kit comparison¶

  • Samples: same UHRR + 1% synthetic ERCC control input (10 ng), prepared with kits from different vendors.
  • Salmon used to generate pseudocounts from transcriptome
  • STAR used to map the reads onto genome

Kit should:¶

  1. Produce clean RNA with high yeild
  2. Be unbiased (or biased?) towards fragment sizes
  3. Carry low rRNA leftovers
  4. Include highest diversity of transcripts and low number of duplicates.

  5. Include High abundance transcripts

  6. Summary

3. Low rRNA content ¶

For this purpose, I aligned the reads to the human genome with STAR and search for rRNAs in a corresponding gtf file. I realized that STAR (with parameters and files that I supplied with) was far from optimal and retrieved a much lower number of isoforms of transcripts for each gene compared to Salmon. Nevertheless, the only purpose of aligning to the genome was to avoid potential missing regions corresponding to rRNAs in the fasta file containing the transcriptome, that has been provided to me.

for kit in ['kit1', 'kit2']:
    # average_kit is the average abundance of duplicates in the sample
    average_kit=[]
    # sub_sample_files lists all splitted files corresponding to a sam file (which correspond to one kit)
    for file in sub_sam_files(kit):
        # open sam file into a list "k" of strings encoding chromosome number, start and length. 
        # Two fragments with equal values are considered duplicates
        k = sam('./STAR_results/'+file)
        # total and unique number of fragments in the sam file
        total = len(k)
        uniq = len(set(k))
        # proporton of duplicate fragments in the sam file.
        average_kit.append((total-uniq) *1. / total)
    # get average of duplicates, considering all sam subsamples
    average_kit = np.sum(average_kit) / len(average_kit)
    print('Fraction of same sequences in {} is {:.3f}'.format(kit, average_kit))

kits.columns = ['kit1', 'kit2']

# total number of samples
total = kits.shape[0]

# make list of 10  bins from high to low abundance and store the bins as sub-indexes in idx 
bins = np.arange(0,total,10000)
idx = [kits.iloc[bins[n]:bins[n+1]].index for n in range(len(bins)-1)]


# list of correlations with blocks of increasing number of samples from high to low abundance
correlations = []

# check correlation between datasets on blocks of high to low abundance transcripts
for n, i in enumerate(idx):
    kits_up = kits.loc[i].values     # updated version of kits including the transcripts belonging to the current block (i)
    q = len(i)                       # total number of transcripts in the block

    # measure correlation of TPMs between kit1 and kit2 for the current block                          
    r = spearmanr(kits_up[:,0], kits_up[:,1])[0]
    R = np.corrcoef(kits_up[:,0], kits_up[:,1])[0][1]
    
    print('for the {} ranked {:.2f}% transcripts with highest TPM, r = {:.3f}'.format(n+1, q*100./total,r))
    correlations.append(r)

Summary: ¶

  • Technical replicates would estimate reproducibility and find possible sources of variability.
  • High or Low abundance transcripts might have been estimated by the vendor of the UHRR and ERCC (the latter should be known). In such case, these estimates should be used to validate the accuracy and precision of the kits.
  • About 30% of transcripts have 0 TPM in both kits (~40% in each kit alone).
  • Any differnce in length of fragments recovered by the kits should be validated with technical replicates.
  • Kit1 seems to carry less rRNA compared to kit2. Again, I don't know what is the expected variation and technical replicates would validate results.
  • Kit2 recovers signal for ~1000 more transcripts copared to kit1 but these transcripts tend to be low abundance.
  • Depending on the library preparation, exact same reads might point to duplications during library preparation. Kit2 might produce more duplicates compared to kit1 (this point might be avoided).
  • TPM for high abundance transcripts tend to correlate between kits. This correlation shows inverse correlation to abundance.
